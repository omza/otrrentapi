Mit otrrent können Nutzer von onlineTVrecorder.com (OTR) per torrent verfügbare beliebteste Aufnahmen von OTR auswählen und die entsprechende torrent Datei an einen heimischen ftp-Server weiterleiten. Am heimischen Rechner kann dann mit einem bittorrent Client die entsprechende Sendung (otrkey) runtergeladen werden.

Mit der Pro Version von otrrent können die Aufnahmen in der Cloud decodiert und geschnitten werden. Dafür benötigt otrrent die OTR Zugangsdaten.
