TOP Aufnahmen

Mit otrrent können Nutzer von onlineTVrecorder.com (OTR), die otrkey Dateien per bittorrent runterladen, eine Sendung aus den beliebtesten Aufnahmen von OTR auswählen und die entsprechende torrent Datei an einen heimischen ftp-Server weiterleiten. Am heimischen Rechner kann dann mit einem bittorrent Client die entsprechende Sendung (otrkey) runtergeladen werden.
