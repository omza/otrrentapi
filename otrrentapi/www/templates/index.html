TOP Aufnahmen

Mit otrrent können Nutzer von onlineTVrecorder.com (OTR) in den beliebtesten Aufnahmen von OTR stöbern und sich Details der Aufzeichnung ansehen. Es werden nur Sendungen angezeigt für die ein Torrent vorhanden ist.

Torrent push

Senden Sie mit einem Klick die Torrentdatei an Ihren FTP-Endpoint. Am heimischen Rechner kann dann mit einem bittorrent Client die entsprechende Sendung (otrkey) runtergeladen werden.

Werbefrei

Nutzen Sie otrrent ohne Werbung! Per In-App Kauf entfernen Sie die Werbeeinblendungen um die beste Nutzererfahrung zu erleben. otrrent blendet in der alpha-Version noch keine Werbung ein!

Decodieren & Schneiden

Werden Sie per In-App Kauf zum Pro-User! Wählen Sie mit Einem-Klick eine Sendung aus. otrrent decodiert und schneidet für Sie den otrkey und sendet das Video an Ihren FTP Endpoint.otrrent bietet in der alpha-Version noch keinen Pro-User Status an!

schritt 3) übergabe der torrent-datei vom ftp-Server an den bittorrent-client

mit einem powershell skript verschiebe ich nun die "*.otrkey.torrent" dateien von einem ftp-server auf das lokale verzeichnis in dem uTorrent die torrent-dateien erwartet. das powershell skript lasse ich dann als zeitgesteuerte aufgabe ausführen. der TechNet artikel "Aufgabenplanung: So wird es gemacht" beschreibt die einrichtung einer Aufgabe. das folgende powershell skript können sie gerne verwenden: 


                    #Download der torrentdateien von einem ftp server
                    #------------------------------------------------------------------------------------------------
                    # Zusammengestellt von app workshop UG (haftungsbeschränkt)


                    #Variablendeklaration
                    #----------------------------------------------------------------------------------------

                    $localTorrentPath = "Austauschverzeichnis mit uTorrent"
                    $ftpserver = "--------"
                    $ftpPath = "/"
                    $ftpFilter = "*.torrent"
                    $ftpUsername = "----------"
                    $ftpPassword = "----------"
                    $ftpserverUrl = $ftpserver + $ftpPath + $ftpFilter

                    cls


                    # Function zum Auslesen
                    #----------------------------------------------------------------------------------------

                    function Receive-Stream {
                    #.Synopsis
                    #  Read a stream to the end and close it
                    #.Description
                    #  Will read from a byte stream and output a string or a file
                    #.Param reader
                    #  The stream to read from
                    #.Param fileName
                    #  A file to write to. If this parameter is provided, the function will output to file
                    #.Param encoding
                    #  The text encoding, if you want to override the default.
                    param( [System.IO.Stream]$reader, $fileName, $encoding = [System.Text.Encoding]::GetEncoding( $null ) )

                    if($fileName) {
                    $writer = new-object System.IO.FileStream $fileName, "Create"
                    } else {
                    [string]$output = ""
                    }

                    [byte[]]$buffer = new-object byte[] 4096
                    [int]$total = [int]$count = 0
                    do
                    {
                    $count = $reader.Read($buffer, 0, $buffer.Length)
                    if($fileName) {
                    $writer.Write($buffer, 0, $count)
                    } else {
                    $output += $encoding.GetString($buffer, 0, $count)
                    }
                    } while ($count -gt 0)

                    $reader.Close()
                    if(!$fileName) { $output }
                    }

                    # Start verarbeitung
                    # FTP Session mit Methode System.Net.FtpWebRequest.ListDirectory
                    #-------------------------------------------------------------------------------------------------------------------------
                    [System.Net.FtpWebRequest] $ftp = [System.Net.WebRequest]::Create("$ftpserverUrl")
                    $ftp.Method = [System.Net.WebRequestMethods+FTP]::ListDirectory
                    $ftp.Credentials = New-Object System.Net.NetworkCredential($ftpUsername, $ftpPassword)

                    $response = $ftp.getresponse()
                    $response.StatusCode

                    #if( $response.StatusCode -ne "ClosingData" ) {
                    #  throw "Response Status: $($response.StatusDescription)"
                    #}


                    $global:ftpOutputCache = (Receive-Stream $response.GetResponseStream())
                    $list = $global:ftpOutputCache -replace "(?:.|\n)*
                    \s+((?:.*|\n)+)\s+

                    (?:.|\n)*",'$1' -split "`n"

                    $response.close
                    $ftp.close

                    # lade torrent dateien im RemoteVerzeichnis  per ftp runter
                    #--------------------------------------------------------------

                    $( foreach($line in $list | ? { $_ }) {

                    # bei fehlern versuche nächstes programm
                    trap {
                    [void]$foreach.moveNext()
                    continue
                    }


                    $urltorrent = $ftpserver + $ftpPath + $line
                    $filetorrent = $localTorrentPath + $line

                    # Zieldatei bereits vorhanden, dann löschen und nochmal runterladen
                    if (Test-Path -Path $filetorrent) {
                    Remove-Item $filetorrent
                    }

                    # Download von FTP
                    $clnt = new-object System.Net.WebClient
                    $clnt.credentials = New-Object System.Net.NetworkCredential($ftpUsername,$ftpPassword)
                    $clnt.DownloadFile($urltorrent,$filetorrent)
                    $clnt.close

                    #ist torrentdatei ordentlich runtergeladen ?

                    if (Test-Path -Path $filetorrent) {

                    #Lösche Remote File
                    # FTP Session mit Methode System.Net.FtpWebRequest.Deletefile
                    #-------------------------------------------------------------------------------------------------------------------------
                    [System.Net.FtpWebRequest] $ftp = [System.Net.WebRequest]::Create("$urltorrent")
                    $ftp.Method = [System.Net.WebRequestMethods+FTP]::Deletefile
                    $ftp.Credentials = New-Object System.Net.NetworkCredential($ftpUsername, $ftpPassword)

                    $response = $ftp.getresponse()
                    $response.StatusCode

                    $response.close
                    $ftp.close

                    }
                    }
                    )

Schritt 4) automatisches dekodieren und schneiden der otrkey-Dateien

                        
                            #Variablendeklaration
                            #----------------------------------------------------------------------------------------
                            $inputPath = "--------"
                            $outputPathPre = "-------"
                            $otrUsername = "--------"
                            $otrPassword = "---------"
                            $otrDecoder = "---------------"
                            $uTorrent = "C:\Users\------\AppData\Roaming\uTorrent\uTorrent.exe"

                            # Stoppe uTorrent
                            #----------------------------------------------------------------------------------------
                            Stop-Process -processname uTorrent* -Force

                            # Über alle Dateien im Decodeverzeichnis
                            #----------------------------------------------------------------------------------------
                            ForEach( $File in Get-ChildItem -Path $inputPath -Filter "*.otrkey")
                            {
                            #pfade bestimmen
                            $inputFile = $File.FullName
                            $outputPath = "none"
                            #Gibt es ein Spezialverzeichnis wie z.B. Homeland?
                            $check = $outputPathPre + "\" + $File.BaseName.split("_")[0]
                            If (Test-Path $check) { $outputPath = $check }
                            Else
                            {
                            #Passt Der, Die Das Verzeichnis
                            $check = $File.BaseName.Substring(0,3)
                            Switch ($check)
                            {
                            "Der" {$outputPath = $outputPathPre + "\_Der"}
                            "Die" {$outputPath = $outputPathPre + "\_Die"}
                            "Das" {$outputPath = $outputPathPre + "\_Das"}
                            }

                            #Sonst anfangsbuchstaben
                            If ($outputPath = "none")
                            {
                            $check = $File.BaseName.Substring(0,1)
                            Switch ($check)
                            {
                            { @("1", "2", "3", "4", "5", "6", "7","8","9","0") -contains $_ } {$outputPath = $outputPathPre + "\_1-9"}
                            { @("a", "A") -contains $_ } {$outputPath = $outputPathPre + "\_A"}
                            { @("b", "B") -contains $_ } {$outputPath = $outputPathPre + "\_B"}
                            { @("c", "C") -contains $_ } {$outputPath = $outputPathPre + "\_C"}
                            { @("d", "D") -contains $_ } {$outputPath = $outputPathPre + "\_D"}
                            { @("e", "E") -contains $_ } {$outputPath = $outputPathPre + "\_E"}
                            { @("f", "F") -contains $_ } {$outputPath = $outputPathPre + "\_F"}
                            { @("g", "G") -contains $_ } {$outputPath = $outputPathPre + "\_G"}
                            { @("h", "H") -contains $_ } {$outputPath = $outputPathPre + "\_H"}
                            { @("i", "I","j", "J") -contains $_ } {$outputPath = $outputPathPre + "\_I-J"}
                            { @("k", "K") -contains $_ } {$outputPath = $outputPathPre + "\_K"}
                            { @("l", "L") -contains $_ } {$outputPath = $outputPathPre + "\_L"}
                            { @("m", "M") -contains $_ } {$outputPath = $outputPathPre + "\_M"}
                            { @("n", "N","o", "O") -contains $_ } {$outputPath = $outputPathPre + "\_N-O"}
                            { @("p", "P","q", "Q") -contains $_ } {$outputPath = $outputPathPre + "\_P-Q"}
                            { @("r", "R") -contains $_ } {$outputPath = $outputPathPre + "\_R"}
                            { @("s", "S") -contains $_ } {$outputPath = $outputPathPre + "\_S"}
                            { @("t", "T") -contains $_ } {$outputPath = $outputPathPre + "\_T"}
                            { @("u", "U","v", "V","w", "W","x", "X","y", "Y","z", "Z") -contains $_ } {$outputPath = $outputPathPre + "\_U-Z"}
                            default {$outputPath = $outputPathPre}
                            }
                            }
                            }
                            #Argumente zusammenbauen

                            $arguments ="-i ""$inputFile"" -o ""$outputPath"" -e ""$otrUsername"" -p ""$otrPassword"" -f -C auto"

                            # rufe decoder auf
                            #Start-Process $otrDecoder -argumentlist $arguments -Wait -verb open

                            # lösche datei
                            Remove-Item $inputFile -recurse -force

                            }

                            #Starte uTorrent
                            #----------------------------------------------------------------------------------------
                            Start-Process $uTorrent
